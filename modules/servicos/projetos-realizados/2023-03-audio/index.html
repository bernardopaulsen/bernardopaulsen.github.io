Diarização de Audio

Neste projeto realizei a diarização de audio de audio de conversas entre n interlocutores. O aplicativo foi desenvolvido em Python.

Aprendizado de Máquina

O modelo de machine learning utilizado para a diarização foi uma rede neural. O modelo foi aplicado com a utilização da biblioteca SpeechBrain, que é baseada na biblioteca PyTorch.
